📖 What is Skitch for Mac?

Skitch is a lightweight screenshot and image annotation application for macOS. It enables users to capture screens or import existing images, then annotate them with visual markup tools before saving or sharing.

Note: Skitch is no longer actively developed by Evernote, but many users continue to use it for its simple interface and fast annotation workflow.
